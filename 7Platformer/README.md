# 🕹️ Platformer – Python Mini Game #7

---

## 🇬🇧 English

Modern-style platformer game built with **Python and Pygame**.

### 🎮 About the game

Jump across platforms, collect coins, avoid enemies and reach the goal.

- Smooth platformer movement 🕹️  
- Jump physics and gravity ⚡  
- Collectible coins 🪙  
- Moving enemy 🤖  
- Level completion system 🚪  
- Modern minimalist visual style ✨  
- Arcade-inspired gameplay 🎮  

### 🚀 How to run

1. Install Python  
2. Install Pygame:

    pip install pygame

3. Run the game:

    python platformer.py

### 🎮 Controls

- A / LEFT ARROW → Move Left  
- D / RIGHT ARROW → Move Right  
- SPACE → Jump  
- ENTER → Restart game  

### 🛠 Technologies

- Python  
- Pygame  

### 📌 Features

- Platformer movement  
- Gravity physics  
- Jump mechanics  
- Coin collection system  
- Enemy movement AI  
- Collision detection  
- Goal / finish system  
- Game over screen  
- Restart system  
- Modern UI design
