# 🎯 Math Shooter – Python Mini Game #8

---

## 🇬🇧 English

Modern arcade-style Math Shooter built with **Python and Pygame**.

### 🎮 About the game

Solve math problems and shoot the correct answer before the targets reach the bottom of the screen.

- Solve random math equations ➕ ➖ ✖️
- Shoot the correct answer 🎯
- Avoid wrong answers ❌
- Multiple lives system ❤️
- Score and level progression 🏆
- Increasing target speed ⚡
- Neon-inspired arcade visuals ✨
- Fast-paced educational gameplay 🚀

### 🚀 How to run

1. Install Python
2. Install Pygame:

    pip install pygame

3. Run the game:

    python math_shooter.py

### 🎮 Controls

- A / LEFT ARROW → Move Left
- D / RIGHT ARROW → Move Right
- SPACE → Shoot
- ENTER → Restart game

- Pygame
