# 🎯 Target Shooting – Python Mini Game #4

---

## 🇬🇧 English

Simple Target Shooting game built with **Python and Pygame**.

### 🎮 About the game

Click moving targets, build combos and survive until the timer runs out.

- Click moving targets 🎯  
- Targets become smaller over time ⚡  
- Target speed increases with levels 🚀  
- Build combo streaks for higher score 🔥  
- Limited misses system 💀  
- Beat your highest score 🏆  

### 🚀 How to run

1. Install Python  
2. Install Pygame:

    pip install pygame

3. Run the game:

    python target_shooting.py

### 🛠 Technologies

- Python  
- Pygame  

### 📌 Features

- Moving targets  
- Increasing difficulty  
- Combo system  
- Score system  
- Level progression  
- Collision detection  
- Timer system  
- Game over screen  
- Restart system
