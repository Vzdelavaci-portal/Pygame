# 🧟 Zombie Survival – Python Mini Game #9

---

## 🇬🇧 English

Modern arcade-style Zombie Survival game built with **Python and Pygame**.

### 🎮 About the game

Survive endless waves of zombies attacking from all directions.

Move, shoot and stay alive as long as possible while the difficulty keeps increasing.

- Zombies attack from every side 🧟
- Aim and shoot using the mouse 🎯
- Increasing difficulty and faster enemies ⚡
- Score and level progression 🏆
- Health system ❤️
- Particle effects and modern visuals ✨
- Fast-paced survival gameplay 🔥

### 🚀 How to run

1. Install Python
2. Install Pygame:

    pip install pygame

3. Run the game:

    python zombie_survival.py

### 🎮 Controls

- W / A / S / D → Move
- Arrow Keys → Move
- Mouse Click → Shoot
- ENTER → Restart game

### 🛠 Technologies

- Python
- Pygame

### 📌 Features

- Top-down survival gameplay
- Zombie AI movement
- Mouse aiming system
- Shooting mechanics
- Particle effects
- Health system
- Score system
- Level progression
- Increasing enemy speed
- Modern arcade UI
- Game over screen
- Restart system
