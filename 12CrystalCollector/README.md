# 💎 Crystal Collector – Python Mini Game #12

---

## 🇬🇧 English

Modern cave adventure game built with **Python and Pygame**.

### 🎮 About the game

Explore a mysterious crystal cave, collect valuable crystals, avoid enemies and use traps strategically to survive.

Build combo chains, earn points and progress through increasingly difficult levels.

- Collect glowing crystals 💎
- Multiple crystal rarities ✨
- Combo scoring system 🔥
- Enemy creatures hunting the player 👾
- Place traps to eliminate enemies 🪤
- Particle effects and visual feedback ⚡
- Procedurally generated cave layout 🗺️
- Increasing difficulty and enemy count 📈

### 🚀 How to run

1. Install Python
2. Install Pygame:

    pip install pygame

3. Run the game:

    python crystal_collector.py

### 🎮 Controls

- W / A / S / D → Move
- Arrow Keys → Move
- E → Place trap
- ENTER → Restart game

### 🛠 Technologies

- Python
- Pygame

### 📌 Features

- Procedurally generated cave
- Collectible crystal system
- Multiple crystal values
- Combo multiplier system
- Enemy AI
- Trap mechanics
- Particle effects
- Score system
- Health system
- Level progression
- Increasing enemy difficulty
- Game over screen
- Restart functionality

### 🎯 Objective

Collect as many crystals as possible, maintain combo chains and survive enemy attacks while progressing through increasingly difficult levels.
