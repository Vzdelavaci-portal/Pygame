# ⚔️ Arena Survivor – Python Mini Game #13

Modern arena survival game built with **Python and Pygame**.

## About the game

Survive endless waves of enemies, collect XP crystals, level up and choose upgrades.

- Automatic magic attacks
- Enemy waves from all directions
- XP collection and level progression
- Upgrade selection menu
- Multiple enemy types
- Particle effects
- Pixel-style PNG assets included
- Modern arcade UI

## How to run

1. Install Python
2. Install Pygame:

    pip install pygame

3. Run the game:

    python arena_survivor.py

## Controls

- W / A / S / D → Move
- Arrow Keys → Move
- 1 / 2 / 3 / 4 → Choose upgrade
- ENTER → Restart after game over
