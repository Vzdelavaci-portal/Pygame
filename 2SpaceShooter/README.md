# 🚀 Space Shooter – Python Mini Game #2

---

## 🇬🇧 English

Simple Space Shooter game built with **Python and Pygame**.

### 🎮 About the game

Control your spaceship, destroy enemies and survive as long as possible.

- Move left and right ⌨️  
- Shoot enemies using SPACE 🔫  
- Avoid collisions with enemies 💀  
- Score points for every enemy destroyed  
- Classic arcade-style gameplay 🚀  

### 🚀 How to run

1. Install Python  
2. Install Pygame:

    pip install pygame

3. Run the game:

    python space_shooter.py

### 🛠 Technologies

- Python  
- Pygame  

### 📌 Features

- Player movement  
- Shooting system  
- Enemy spawning  
- Collision detection  
- Score system  
- Game over screen  
- Restart system  
- Simple arcade gameplay
