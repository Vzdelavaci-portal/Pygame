# ⚔️ Dungeon Crawler V1.1

Modern fantasy dungeon crawler built with **Python** and **Pygame**.

Explore procedurally generated dungeons, defeat dangerous enemies, collect treasure, level up your hero and descend into increasingly challenging dungeon floors.

# ✨ Features

- ⚔️ Sword combat
- 🏰 Procedurally generated dungeon
- 👾 Two enemy types
  - Slime
  - Skeleton
- 💰 Collect gold
- 📦 Treasure chests
- ✨ Particle effects
- 💥 Screen shake on hit
- 🌑 Dynamic lighting (Fog of War)
- 🌀 Portal to the next floor
- ❤️ Character leveling
- 🎁 Upgrade selection
- 📈 Increasing difficulty
- 🧱 Wall collision
- 🎮 Smooth player movement

---

# 🎮 Controls

| Key | Action |
|------|--------|
| **W A S D** | Move |
| **Arrow Keys** | Move |
| **SPACE** | Sword attack |
| **E** | Open chest |
| **1 / 2 / 3** | Choose upgrade |
| **ENTER** | Restart after death |

---

# ❤️ Level System

Defeat enemies to gain experience points.

Every level lets you choose one upgrade:

- ⚔️ Damage
- ❤️ Maximum Health
- 👟 Movement Speed

---

# 👾 Enemies

## 🟢 Slime

- Slow movement
- Low health
- Easy to defeat

---

## 💀 Skeleton

- Faster movement
- More health
- More dangerous

---

# 📦 Treasure Chests

Treasure chests may contain:

- 💰 Gold
- ❤️ Health
- 💎 Large treasure reward

---

# 🌀 Next Floor

After defeating every enemy, a magical portal appears.

Walk into the portal to generate a brand new dungeon floor with increased difficulty.

# ▶️ Installation

Install Pygame:

```bash
pip install pygame
```

Run the game:

```bash
python dungeon_crawler.py
```

- 👟 Rychlost pohybu
