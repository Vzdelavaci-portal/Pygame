# 🦘 Jump Game – Python Mini Game #6

---

## 🇬🇧 English

Simple Jump Game built with **Python and Pygame**.

### 🎮 About the game

Jump over obstacles, survive as long as possible and beat your high score.

- Jump over incoming obstacles 🦘  
- Increasing game speed over time ⚡  
- Endless runner gameplay 🎮  
- Score and level system 🏆  
- Simple arcade mechanics 🕹️  
- Fast reflex gameplay 🔥  

### 🚀 How to run

1. Install Python  
2. Install Pygame:

    pip install pygame

3. Run the game:

    python jump_game.py

### 🎮 Controls

- SPACE → Jump  
- ENTER → Restart game  

### 🛠 Technologies

- Python  
- Pygame  

### 📌 Features

- Jump mechanics  
- Gravity physics  
- Random obstacle spawning  
- Increasing difficulty  
- Score system  
- Level progression  
- Collision detection  
- Game over screen  
- Restart system
