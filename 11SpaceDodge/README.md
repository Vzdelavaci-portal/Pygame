# 🚀 Space Dodge – Python Mini Game #11

---

## 🇬🇧 English

Fast-paced space survival game built with **Python and Pygame**.

### 🎮 About the game

Pilot your spaceship through an asteroid field and survive as long as possible.

The longer you survive, the faster and more dangerous the game becomes.

- Dodge incoming asteroids ☄️
- Endless survival gameplay 🚀
- Increasing difficulty and speed ⚡
- Score based on survival time 🏆
- Level progression system 📈
- Space-themed visuals 🌌
- Simple controls, challenging gameplay 🎮

### 🚀 How to run

1. Install Python
2. Install Pygame:

    pip install pygame

3. Run the game:

    python space_dodge.py

### 🎮 Controls

- W / A / S / D → Move spaceship
- Arrow Keys → Move spaceship
- ENTER → Restart game

### 🛠 Technologies

- Python
- Pygame

### 📌 Features

- Space survival gameplay
- Asteroid spawning system
- Dynamic difficulty scaling
- Score system
- Level progression
- Collision detection
- Animated star background
- Modern arcade UI
- Game over screen
- Restart system
