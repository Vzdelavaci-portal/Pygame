# 🏓 Pong – Python Mini Game #5

---

## 🇬🇧 English

Classic Pong game built with **Python and Pygame**.

### 🎮 About the game

Play against the computer or another player in this arcade-style Pong game.

- Player vs Computer mode 🤖  
- Player vs Player mode 👥  
- Multiple difficulty levels ⚡  
- Different winning score options 🏆  
- Smarter AI movement 🎯  
- Adjustable ball speed 🚀  
- Retro arcade gameplay 🕹️  

### 🚀 How to run

1. Install Python  
2. Install Pygame:

    pip install pygame

3. Run the game:

    python pong.py

### 🎮 Controls

#### Player 1
- W → Move Up  
- S → Move Down  

#### Player 2
- ↑ → Move Up  
- ↓ → Move Down  

### 🛠 Technologies

- Python  
- Pygame

- Multiple game modes  
- Difficulty selection

- Ball physics

- Score system  
- Game over screen  
- Restart system
