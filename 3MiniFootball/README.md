# ⚽ Mini Football – Python Mini Game #3

---

## 🇬🇧 English

Simple Mini Football game built with **Python and Pygame**.

### 🎮 About the game

Control the player, bounce the ball and try to survive as long as possible.

- Move left and right ⌨️  
- Bounce the ball back into the field ⚽  
- Score points for every successful hit 🎯  
- Don't let the ball fall down 💀  
- Simple arcade-style gameplay 🎮  

### 🚀 How to run

1. Install Python  
2. Install Pygame:

    pip install pygame

3. Run the game:

    python mini_football.py

### 🛠 Technologies

- Python  
- Pygame  

### 📌 Features

- Player movement  
- Ball physics  
- Collision detection  
- Score system  
- Game over screen  
- Restart system  
- Simple football arcade gameplay
