# 🏙️ Tower Stack – Python Mini Game #10

---

## 🇬🇧 English

Modern Tower Stack game built with **Python and Pygame**.

### 🎮 About the game

Build the tallest tower possible by stacking moving blocks on top of each other.

Perfect timing is the key to success. Every misplaced block becomes smaller, making the game progressively harder.

- Stack moving blocks 🏗️
- Precise timing gameplay ⏱️
- Increasing difficulty ⚡
- Dynamic tower building 🏙️
- Camera follows the growing tower 📈
- Beautiful city skyline background 🌇
- Score based on completed floors 🏆

### 🚀 How to run

1. Install Python
2. Install Pygame:

    pip install pygame

3. Run the game:

    python tower_stack.py

### 🎮 Controls

- SPACE → Place block
- Mouse Click → Place block
- ENTER → Restart game

- Pygame
