# 🌊 Ocean Explorer V2 – Python Mini Game #14

Atmospheric underwater exploration game built with **Python and Pygame**.

## 🇬🇧 English

### 🎮 About the game

Pilot a small submarine, explore the ocean depths, collect treasures, manage oxygen and return to the surface to store your gold.

Version 2 adds upgrades, new treasures, shark enemies, oxygen vents and darker ocean zones.

### ✨ V2 Features

- Submarine exploration
- Oxygen management
- Treasure collection
- Stored gold system
- Upgrade station at the surface
- Oxygen Tank upgrade
- Engine upgrade
- Spotlight upgrade
- Hull upgrade
- New treasures: Ruby, Sapphire, Ancient Artifact
- Oxygen vents
- Piranhas
- Jellyfish slowing effect
- Shark enemy with dash behavior
- Dynamic ocean zones
- Real darkness based on depth
- Upgradeable spotlight
- Bubble effects
- Screen shake
- PNG assets included

### 🚀 How to run

1. Install Python
2. Install Pygame:

    pip install pygame

3. Run the game:

    python ocean_explorer.py

### 🎮 Controls

- W / A / S / D → Move submarine
- Arrow Keys → Move submarine
- U → Open upgrades at surface
- 1 / 2 / 3 / 4 → Buy upgrade
- ESC → Close upgrade menu
- ENTER → Restart after game over
