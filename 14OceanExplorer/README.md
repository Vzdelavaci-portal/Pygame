# 🌊 Ocean Explorer – Python Mini Game #14

---

## 🇬🇧 English

Atmospheric underwater exploration game built with **Python and Pygame**.

### 🎮 About the game

Pilot a small submarine into the depths of the ocean.

Collect treasures, manage your oxygen supply, avoid dangerous sea creatures and return safely to the surface to secure your discoveries.

The deeper you dive, the darker and more dangerous the ocean becomes.

### 🌊 Features

- Explore increasingly deeper ocean zones
- Oxygen management system
- Treasure collection
- Multiple treasure rarities
- Dangerous sea creatures
- Piranhas and jellyfish
- Temporary slowing effects
- Dynamic depth progression
- Treasure banking system
- Bubble particle effects
- Dynamic underwater atmosphere
- Submarine spotlight
- Difficulty scaling with depth
- Modern arcade-style UI

### 💎 Treasure Types

#### Chest
- Value: 10 Gold

#### Pearl
- Value: 20 Gold

#### Ancient Relic
- Value: 50 Gold

#### Lost Artifact
- Value: 100 Gold

#### Oxygen Capsule
- Restores oxygen

### 🐟 Enemies

#### Piranha
- Fast predator
- Damages the submarine

#### Jellyfish
- Electric creature
- Temporarily slows the submarine

### 🌊 Ocean Zones

#### Surface (0-100m)
- Bright water
- Safe exploration

#### Deep Ocean (100-300m)
- Darker environment
- More treasures

#### Abyss (300m+)
- Dangerous depths
- Increased difficulty

### 🚀 How to run

1. Install Python

2. Install Pygame

```bash
pip install pygame
```

3. Run the game

```bash
python ocean_explorer.py
```

### 🎮 Controls

- W → Move Up
- S → Move Down
- A → Move Left
- D → Move Right

or

- Arrow Keys → Move Submarine

### 🎯 Objective

- Dive deeper into the ocean
- Collect treasures
- Monitor oxygen levels
- Avoid sea creatures
- Return near the surface
- Store your treasure safely
- Beat your previous record

- Python
- Pygame

## 🇨🇿 Česky

Atmosférická podmořská hra vytvořená pomocí **Pythonu a Pygame**.

### 🎮 O hře

Ovládej malou ponorku a prozkoumávej hlubiny oceánu.

Sbírej poklady, hlídej si zásobu kyslíku, vyhýbej se nebezpečným mořským tvorům a bezpečně se vracej na hladinu, abys uložil nalezené poklady.

Čím hlouběji se dostaneš, tím temnější a nebezpečnější oceán bude.

### 🌊 Funkce

- Průzkum stále hlubších částí oceánu
- Systém kyslíku
- Sbírání pokladů
- Více druhů pokladů
- Nebezpeční mořští tvorové
- Piraně a medúzy
- Efekt zpomalení po zásahu medúzou
- Dynamická hloubka a obtížnost
- Ukládání pokladů po návratu na hladinu
- Bublinkové particle efekty
- Atmosférické podmořské prostředí
- Reflektor ponorky
- Obtížnost rostoucí s hloubkou
- Moderní arkádové UI
